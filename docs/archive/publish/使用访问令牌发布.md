# 使用访问令牌（Access Token）发布

如果不想设置 2FA，可以使用访问令牌发布。

## 步骤 1: 创建访问令牌

1. 在浏览器中打开：https://www.npmjs.com/settings/codegank/tokens
2. 点击 "Generate New Token"
3. 选择 "Classic Token"
4. 选择权限：
   - **Automation** - 用于 CI/CD（推荐用于发布）
   - 或 **Publish** - 仅用于发布包
5. 复制生成的令牌（只显示一次！）

## 步骤 2: 使用令牌登录

在命令行中运行：

```bash
npm logout
npm login
```

当提示输入密码时，粘贴你的访问令牌。

## 步骤 3: 发布

```bash
npm publish --access public
```

---

## 或者：直接使用令牌发布

不登录，直接使用令牌：

```bash
npm publish --access public --//registry.npmjs.org/:_authToken=YOUR_TOKEN_HERE
```

将 `YOUR_TOKEN_HERE` 替换为你的令牌。

---

**注意**：
- 令牌只显示一次，请妥善保存
- 不要将令牌提交到 Git
- 定期更换令牌以保证安全
